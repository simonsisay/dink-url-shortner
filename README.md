# dink-url-shortner

Frontend:React, Next.js, Apollo-Client.            
Backend: Node, Objection.js, Graphql &amp; Apollo-server

To run the project

cd into the frontend directory
run 

yarn or npm install

yarn run dev


cd into backend directory

run yarn 

yarn start will start the server
